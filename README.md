# SUN StopWatch☀
> 이 프로젝트는 2020년도 웹 프로그래밍 기초 과목의 2학기 수행평가 산출물로 제작 되었습니다.

#### - 사용 언어📖
> javascript, html, css


1. <b>주요 기능<b/>
* 시간, 분, 초, 100/1초 측정⏱
* LAP을 통해 시간 기록✏
* 시간 초기화💨
# 
2. <b>추가 기능<b/>
* 키보드를 통한 버튼 조작💻
* 기록된 시간의 최소, 최대, 평균 시간을 계산📑
* 반응형 UI📱
